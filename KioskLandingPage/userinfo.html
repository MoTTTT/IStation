Quick Start

logging in:

  • when the windows system start up a clear screen will be displayed with a login window
  • click with the mouse in the "login" textfield and enter the user name by typing on the keyboard i.e. type "browser"
  • click with the mouse in the "password" textfield and enter the user password for browser i.e. type in  "oracle", note that no text will be displayed in this textfield as you type
  • click on the "Options..."  button and select the default option (if selected the displayed button would appear sunk)
  • either press the "Enter" key on the keyboard or click on the "login" button

mount stiffy:

  • unlike some other operating systems such as Windows 98, a stiffy must be "mounted" on the file system before it can be used (this feature allows the user to mount the stiffy anywhere in the directory tree of the pc - the default mount is set up as /mnt/floppy ).
  • the stiffy can be mounted by double clicking with the left mouse button on the "Floppy 0" icon on the desktop. This will bring up the Gnome "File Manager" window displaying the files on the stiffy now mounted under the folder /mnt/floppy on the directory tree of the pc
        or
    right click with the mouse on the "Floppy 0" icon on the desktop
    • a menu would be displayed
    • select the "Mount device" option (if the "Unmount device" option is displayed, then a stiffy has been removed without first unmounting it, first select the "Unmount device" option before proceeding to mount your own stiffy).
    • To view the the files on the stiffy now mounted under the folder /mnt/floppy on the directory tree of the pc, right click again on the "Floppy 0" icon and select the "Open" option.

web browsing with Netscape Communicator:

  • the Gnome menu is brought up by clicking on the Gnome icon (footprint) on the task bar at the bottom of the screen. Select the "Internet" menu item. On the Internet sub-menu select "Netscape Communicator".
or
    click on the Netscape Communictor icon on the task bar at the bottom of the screen.
     
  • to surf the web the communication link (telephone line) must be opened to the ISP (Internet Service Provider). In order to do this click on the Gnome icon on the task bar at the bottom of the screen.
    • Point the mouse at the "Another level" menu item. On the "Networking" sub menu which will appear, select the "Usernet" menu item. A "UserNet" window will appear.
    • On this window two buttons will be displayed. The button marked "PPP0" is the button used to toggle the communications link on/off. The other button is just used to display the status of the communication link:
      • RED means that the communication link to the web is closed and the user does not have access to the web
      • GREEN means that the communication link to the web is open and the user can surf the web
      • YELLOW is an intermediate state when the pc is in the process of dialing out to the ISP
    • to surf the web the link must be open i.e. the status button must be GREEN, if the status button is RED, click on the PPP0 button once, you will hear the pc dial out, the status button will change its colour to YELLOW, when a link has been established the status button will change to GREEN.   If the status button is YELLOW  do not click on the PPP0 button.

hotmail:

  • open the communication link to the Internet.
  • in Netscape Communicator click in the "location:"  textfield at the top of the application window and enter "http://www.hotmail.com", press the "Enter" key on the keyboard.
  • the "login" page of hotmail will be displayed. A hotmail account can be opened by clicking on "Sign Up Here" and following the instructions.
  • To read and send mail:
    • on the hotmail login page enter your username in the "Login Name" textfield and your password in the "Password" textfield, click with the mouse on the "Enter" button. A Netscape Security warning window would appear, click the "Continue Submission" button.
    • the "Inbox" of your hotmail account will be displayed. All new messages will be displayed on this page. To read a message click on the displayed sender name in the "from" column, the selected e-mail messages would be loaded into the web page. To reply to this message, click on the "reply" button displayed above the message.
    • to send an e-mail message click on the "Compose" menu item displayed on the top left hand side of the hotmail screen.
      • enter the e-mail address of the person you want to send the message to in the "To:" textfield.
      • enter the topic of the message in the "Subject:" textfield.
      • click inside the "textarea" in the middle of the page and start to type in the message you want to send. When finished, and you are ready to send the message, click on the "Send" button at the top of the screen.

editing web pages:

  •  in Netscape Navigator select the menu item "Open Page" in the "File" menu at the top of the screen. An "Open Page" window will appear, left click the "Choose File ..." button. A window will be displayed allowing the user to navigate the directory structure of the pc and select the appropriate file to be edited. The desired file is selected by clicking on it and selecting the "Open" button on the window (selecting the "Cancel" button will abort the operation).
  • the selected file will be displayed in the "Open" window textfield. To edit the contents of the file select the "Composer" radio button ("Navigator" is the Netscape browser which will allow the selected file to be viewed but not edited) and click the "Open" button.
  • the opened file can now be edited, Composer has the same functionality as a basic text editor .
  • to create a hyperlink,
    • left click on the text which is to serve as the link.
    • drag the mouse while holding down the left button to highlight the desired text. Right click on the highlighted text, a pop-up menu will be displayed.
    • select the menu item "Create Link Using Selected...". A "Character Properties" window will be displayed. In the "Link" tabpanel click on the "Choose File..." button.
    • the directory structure of the pc can now be navigated in the "Link to File" window and the appropriate file selected. Click "Open", the selected file will be displayed in the "link to page location or local file:" textfield. Select the "OK" button to enable the link.
    • the linking text in the document will be underlined and printed in a different colour from the rest of the text in the document.
    • when viewing the document in a web browser such as Netscape Navigator the user can by clicking on the linked hypertext navigate to the file to which it is linked.
  • to insert an image into the webpage click on the "Insert" item on the menu-bar at the top of the Netscape Composer screen.
    • click on the "Image.." menu item in the pop-up menu that would be displayed.
    • select the "Choose File..."  button in the "Image Properties" window. Navigate the directory structure of the pc to select the image that you want to display in the webpage. Click "Open" to select the desired image.
    • the absolute location of the file would be displayed in the "Image location - Enter a remote URL or local file" textfield, edit the displayed URL if a relative URL for the image is required (advisable) e.g. the absolute URL is "file:///G|/qsolutio/webpage/image.jpg" if this file is located in the same directory as the webpage in which this image is to be displayed then edit the URL to be "image.jpg".
