Trainer Notes
 
  • Introduction
    • open modem connection
    • explain internet and that the pc is connected to the net via a telephone link to the ISP's computer (server) that is permanently connected to the internet
    • briefly show the desktop environment, movement of the mouse, menus, menubar, minimise and maximise, close
  • Netscape
    • webbrowser - it is a tool used to view webpages on the internet, webpages are hypertext files with links to other pages resulting in a web of interlinked documents. If you click on a piece of text or a picture in one page you are immediately transfered to some point either on the same page or some other document which can sit on the other side of the world.
    • start Netscape on an open desktop screen.
    • start up in Home, explain Home
    • navigate the pages in Home to give a basic explanation of links, Forward, Back buttons
    • Print a page, explain the settings.
    • show the bookmarks, goto one of the bookmarks, indicate the changing of the URL address
    • type in an adress e.g. http://www.ananzi.co.za , explain the set up of the address.
    • explain the importance of search engines
    • click on Search button
    • select the Lycos search engine
    • want to search for the Labia cinema
    • enter: +cinema +labia +cape +town
    • explain the use of:    +    -    " "    AND    OR
    • follow the results of the search to the Labia home page: www.labia.co.za
    • save an image to the hard drive e.g. the street map for the labia cinema
    • mount a stiffy on the machine - double click on the stiffy icon on the screen
    • save the image from the hard drive to the stiffy by dragging and dropping
    • unmount the stiffy
