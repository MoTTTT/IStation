Q Solutions Welcomes You To KioX, Your Linux Information Kiosk

The Q Solutions Linux Information Kiosk is a PC preloaded with WWW server and browser to provide a container for the user friendly content of browser based technologies.
A closed set of files with links and their objects are loaded up to the kiosk, to form a contained hypertext document. If the multimedia option is taken, these can include sound. If the touch screen is chosen, the unit can be used without the mouse and keyboard, to form a robust solution ready for the field.

Design Goals:

  • Configurable
  • Low maintenance
  • Rapid deployment
  • Robust
Software:
  • Browser: Netscape
  • WWW Server: Apache
  • Operating System: Linux
  •  X Window System: XFree86
Hardware Options:
  • Multimedia: A Sound Card adds Audio support to your web pages
  • Touch screen
  • Remote access modem
