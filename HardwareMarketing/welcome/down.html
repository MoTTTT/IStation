Shutting Down the Host

It should not be necessary to reboot your Linux PC, or to switch it off. If you would like to do so in order to move it, or add new hardware, you may log in as "root" on either the GUI or text console, and issue the command "reboot" on the command line. (In the case of the GUI interface, you should start up an xterm to do this.) An alternative is to press <CTRL><ALT><DEL> while in the text console.
