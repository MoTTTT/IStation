Virtual Consoles

Four virtual consoles may be viewed.

  • Console 1 shows the system startup messages, ending with the startup message for the X Windows System.
  • Console 2 contains the X Windows Graphical User Interface (GUI)- the one you are viewing now.
  • Console 6 provides a text console which you can use by logging in at the prompt.
  • Console 8 shows system messages.

An alternative console can be used by pressing <CTRL><ALT><FConsoleNumber>, where FConsoleNumber is F1, F2, F6, or F8. For example, to view the system logs, use <CTRL><ALT>F8, and to return to the X Windows GUI, use <CTRL><ALT>F2.
