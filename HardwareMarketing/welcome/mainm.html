The Window Manager's Menus

A number of utilities are available via the Window Manager's menu.

The Main Menu is accessed by:

  • Clicking with the left mouse button on the desktop (i.e. the green background on which the windows are drawn.)
  • Clicking on the button on the left hand side of your Task Bar, and selecting Main Menu.The Start button also gives immediate access to a few of the Main Menu's options.

By selecting an option in the Main Menu, programs can be started, or sub-menus opened. Items in the Main Menu with a triangle in the right margin are sub-menus.

The following programs can be started directly from the Main Menu:

  • "Midnight Commander" a "Norton Commander" like file manager;
  • "Top", a dynamic display of system statistics and processes running on your host;
  • "Manual Pages", an invaluable resource detailing the usage of UNIX and most of the programs loaded on your hard drive,
  • The "Shell" option provides a large color xterm window for entering Unix commands.
  • The "OClock" item creates a large analog clock which is displayed on the desktop.

The first set of sub menus provide general programs in the following groups:

  • Applications,
  • Games,
  • Internet Utilities, and
  • Graphics Utilities.

The next set provides access to peripherals, screen related utilities, and window manager utilities:

  • Sound: If you have a sound card and CD ROM drive, you can play DC's using Workman, and XMix.
  • Floppy: You can read and write files on a floppy disk after mounting the drive using this menu.
  • Screen: A variety of screen savers, and screen locks are available from this sub-menu.
  • Modules: Access to a few window manager utilities, such as the Task Bar, and the Button Bar.

The Refresh Screen option will redraw the desktop, and all the windows on it.

The Exit sub-menu provides the choice of restarting the window manager, starting an alternative window manager, or shutting down the window manager. The latter has the effect of shutting down all running programs, and logging the user off from the GUI Server. The GUI Server login screen indicates that a new session can be started.
